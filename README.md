# production_schedule

## Project setup

```bash
yarn
```

### Install JSON Server

```bash
yarn global add json-server
```

### Start JSON SERVER

```bash
json-server --watch db.json
```

### Compiles and hot-reloads for development

```bash
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
